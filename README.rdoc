== SPINDI TRANSLATE (СПИНДЕРМААН ПРЕВОДАЧ)

SpindiTranslate is a web app which translates from/to Spindermaan's language.

Specifications:

* Ruby 2.1.2

* Rails 4.2.4

* PostgreSQL database

* Bootstrap theme 'Freelancer' - http://startbootstrap.com/template-overviews/freelancer/


Deployed at http://spinditranslate.herokuapp.com
